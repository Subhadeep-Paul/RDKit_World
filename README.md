
# RDKit_World

RDKit_World is made for those who wants to learn the practical aspects of RDKit for using in AI/ML in chemistry & cheminformatics.


## Tech Stack

**Language:** Python

**Libraries:** NumPy, Pandas, Seaborn, RDKit
